Restaurant Queue Management System

Streamline your walk-in experience with a simple, real-time queue.

RQMS is a lightweight web application built for restaurant staff to manage guest waiting lists efficiently. No more paper lists or manual tracking — just a clean dashboard that keeps your front-of-house running smoothly.

Key features:

  • Secure staff registration and login
  • Add guests to the live queue with name, phone, and party notes
  • Mark guests as Seated, Done, or Cancelled in one click
  • Live table management — track availability, occupancy, reservations
  • Queue stats dashboard — waiting, seated, done, cancelled counts
  • Customers can self-join the queue after logging in

Built with Go (Gorilla Mux), PostgreSQL, and plain HTML / CSS / JavaScript — MVC architecture, Docker-ready deployment.
